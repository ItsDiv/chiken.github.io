# 치느님의 이야기 - 어린이 만화 웹사이트 🐔✨

치경의 이야기를 귀여운 만화 스타일로 보여주는 어린이 웹사이트입니다.

## 📁 파일 구조

```
치느님/
├── index.html          # 메인 HTML 파일
├── styles.css          # 스타일시트
├── script.js           # JavaScript 파일
├── images/             # 이미지 폴더
│   ├── chapter1-1.jpg
│   ├── chapter1-2.jpg
│   ├── chapter1-3.jpg
│   ├── chapter1-4.jpg
│   ├── chapter2-1.jpg
│   ├── chapter2-2.jpg
│   ├── chapter2-3.jpg
│   ├── chapter2-4.jpg
│   ├── chapter3-1.jpg
│   ├── chapter3-2.jpg
│   ├── chapter3-3.jpg
│   ├── chapter3-4.jpg
│   ├── chapter4-1.jpg

│   ├── chapter4-3.jpg
│   └── chapter4-4.jpg
└── README.md           # 이 파일
```

## 🎨 주요 기능

### 1. 챕터 네비게이션
- 4개의 치경 장을 버튼으로 쉽게 전환
- 화살표 키(←, →)로도 이동 가능
- 부드러운 페이드인 애니메이션

### 2. 반응형 디자인
- 모바일, 태블릿, 데스크톱 모두 지원
- 화면 크기에 따라 자동 조정

### 3. 인터랙티브 효과
- 카드 호버 시 확대 효과
- 스크롤 애니메이션
- 플로팅 아이콘 애니메이션

### 4. 이미지 지원
- 각 구절마다 이미지 표시
- 이미지가 없을 경우 귀여운 아이콘 플레이스홀더 표시

## 🖼️ 이미지 추가 방법

`images` 폴더에 다음 파일명으로 이미지를 추가하세요:

**1장 - 닭과 알의 비밀**
- `chapter1-1.jpg` - 하늘을 보며 고민하는 사람들
- `chapter1-2.jpg` - 바람에 실려가는 질문
- `chapter1-3.jpg` - 치느님이 알을 창조하는 모습
- `chapter1-4.jpg` - 알에서 나온 첫 닭

**2장 - 세상의 창조**
- `chapter2-1.jpg` - 달걀껍질이 흙이 되는 모습
- `chapter2-2.jpg` - 알껍질이 태양이 되는 모습
- `chapter2-3.jpg` - 산맥이 만들어지는 모습
- `chapter2-4.jpg` - 강과 풍성한 자연

**3장 - 인간의 탄생**
- `chapter3-1.jpg` - 생각에 잠긴 치느님
- `chapter3-2.jpg` - 원숭이가 변화하는 모습
- `chapter3-3.jpg` - 말을 배우는 모습
- `chapter3-4.jpg` - 첫 인간의 탄생

**4장 - 치킨의 축복과 심판**
- `chapter4-1.jpg` - 치킨을 받는 인간들
- `chapter4-2.jpg` - 치킨을 숨기는 인간들
- `chapter4-3.jpg` - 메마른 땅
- `chapter4-4.jpg` - 치킨이 귀해진 세상

## 🚀 실행 방법
